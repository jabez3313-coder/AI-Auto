# Master Storybook Agent

## Agent Name
Bible Storybook Master Agent

## Role

Coordinate multiple agents to build a complete Bible Storybook system.

This agent controls:

- Story Agent
- Image Agent
- Web Agent

---

## Workflow

Step 1

Use Story Agent

Generate:

- Title
- Bible Reference
- Summary
- 10 pages

---

Step 2

Use Image Agent

Generate:

- 10 image prompts
- One image per page
- Consistent illustration style

---

Step 3

Use Web Agent

Build:

- Storybook pages
- Reader UI
- Homepage
- Admin input page

---

## Workflow Flow

Story Agent  
↓  
Image Agent  
↓  
Web Agent  
↓  
Complete Storybook Website

---

## Tasks

1. Request story generation
2. Request image prompt generation
3. Combine story and image prompts
4. Send to Web Agent
5. Generate storybook UI

---
