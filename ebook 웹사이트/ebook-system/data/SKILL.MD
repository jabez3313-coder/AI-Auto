# Data Folder Skill

This folder stores story data.

Each story is stored as JSON.

Example:

{
 "title": "Story Title",
 "pages": [
   {
     "text": "Page text"
   },
   {
     "image": "image.jpg",
     "text": "Page text"
   }
 ]
}

Rules:
- Flexible page structure
- Optional image
- Optional text
- Optional audio
- Optional video

Naming rules:
story-id.json

Example:
peter-01.json
jesus-01.json
