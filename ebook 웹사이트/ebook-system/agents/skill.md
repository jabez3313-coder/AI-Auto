# Agent Skill

Agents generate content.

Agents:
- Story Agent
- Image Agent
- eBook Agent

Story Agent:
- Create 10 page story
- Maintain tone
- Maintain character consistency

Image Agent:
- Generate images
- Keep consistent style
- Match story text

eBook Agent:
- Combine story and image
- Create JSON
- Save story

Rules:
- Warm tone
- Storybook style
- Consistent characters
