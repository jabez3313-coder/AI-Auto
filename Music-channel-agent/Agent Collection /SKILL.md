# Music Channel Agent System

## Overview

This repository contains a coordinated system of AI agents designed to automate the planning, creation, packaging, and publishing preparation of music content for a YouTube music channel.

Each agent has a specialized role. Together they form a complete workflow that turns an idea into a ready-to-publish music release package.

The system focuses on Suno-based music creation, YouTube optimization, and scalable channel production.

---

# Agent Architecture

The system includes the following agents:

1. strategist
2. song-planner
3. style-generator
4. lyrics-maker
5. suno-prompt-maker
6. suno-planner
7. description-generator
8. thumbnail-generator
9. packaging-agent
10. operations-agent

---

# Workflow Overview

The agents collaborate in a structured production pipeline.

Idea → Song Planning → Style → Lyrics → Suno Prompt → Generation Plan → Description → Thumbnail → Packaging → Operations

---

# Production Pipeline

## 1. Strategist

Defines the overall channel direction and long-term content strategy.

Responsibilities:

* Define channel identity
* Plan content themes
* Identify target audience
* Suggest content series and trends

---

## 2. Song Planner

Designs the concept of each song before production.

Responsibilities:

* Define song theme and message
* Plan emotional flow
* Determine audience context
* Create a song blueprint for other agents

---

## 3. Style Generator

Defines the musical style and sonic direction.

Responsibilities:

* Suggest genre and subgenre
* Define mood and atmosphere
* Propose instrumentation
* Recommend BPM and arrangement feel

---

## 4. Lyrics Maker

Creates original lyrics aligned with the song plan.

Responsibilities:

* Write lyrics
* Structure verse, chorus, bridge
* Match emotional tone and audience

---

## 5. Suno Prompt Maker

Transforms song ideas into effective Suno prompts.

Responsibilities:

* Convert song concept into Suno input prompts
* Define musical details such as genre, vocal type, tempo, and mood
* Generate variations for testing

---

## 6. Suno Planner

Designs the generation workflow for Suno.

Responsibilities:

* Plan generation order
* Suggest prompt testing strategy
* Organize version comparisons
* Optimize production efficiency

---

## 7. Description Generator

Creates engaging descriptions for YouTube uploads.

Responsibilities:

* Write song introductions
* Create playlist descriptions
* Communicate mood and purpose of the track
* Encourage listener engagement

---

## 8. Thumbnail Generator

Designs the visual concept for the video thumbnail.

Responsibilities:

* Create thumbnail concepts
* Generate image prompts for visual tools
* Ensure strong click appeal
* Align visuals with song mood

---

## 9. Packaging Agent

Combines all assets into a final release package.

Responsibilities:

* Assemble title, description, thumbnail concept, and song context
* Prepare upload-ready content package
* Ensure brand consistency

---

## 10. Operations Agent

Manages the workflow and operational structure of the channel.

Responsibilities:

* Coordinate production steps
* Organize assets and file structures
* Maintain repeatable workflows
* Support scalable production

---

# Design Principles

The system follows these principles:

Consistency
All songs and visual assets should align with the channel identity.

Efficiency
The workflow should minimize wasted generation attempts.

Scalability
The structure should support batch production of music content.

Clarity
Each agent has a clearly defined responsibility.

---

# Final Output

The complete agent pipeline produces a ready-to-publish music package including:

Song concept
Lyrics
Suno prompt
Generated music plan
Thumbnail concept
YouTube description
Release packaging

This allows the music channel to scale production efficiently while maintaining creative quality.
