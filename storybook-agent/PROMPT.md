# Bible Storybook Agent Prompt

Build an MVP web app for a Bible Storybook service.

The MVP should NOT include video.
It should only support:
- 10-page storybooks
- one image per page
- short text per page
- responsive web viewing

Goal:
Create a warm, simple, child-friendly web app where users can read Bible stories as 10-page digital storybooks.

Core Features:

## 1. Home Page
- Service title
- Subtitle
- Storybook list
- Card layout
- Cover image
- Title
- Bible reference
- Summary
- Read button

## 2. Story Reader Page
Each story must contain:

- 10 pages
- 1 image per page
- short text
- page number

Navigation:
- Next button
- Previous button
- Back button
- Restart button on last page

## 3. Admin Page

Allow input:

- Title
- Bible Reference
- Summary
- Cover image URL
- 10 pages

Each page:

- page number
- image URL
- short text

## Design Style

- Warm
- Soft
- Clean
- Child friendly
- Rounded UI
- Large text

## Tech Requirements

- React or Next.js
- Local JSON first
- No Firebase yet
- Easy future upgrade

## Components

- Header
- Footer
- StoryCard
- StoryList
- StoryReader
- PageView
- AdminForm

## Sample Stories

Create 3 sample stories

- Noah’s Ark
- David and Goliath
- Good Samaritan

## Output

Generate:

- folder structure
- pages
- components
- sample data
- working MVP
- run instructions

Important:

No video

No payment

Only Storybook MVP
