# eBook System Skill

This project builds a web-based eBook story system.

Goals:
- Create storybooks
- Display storybooks
- Support flexible content (text, image, audio, video)

Story Structure:
- 10 pages default
- Each page optional:
  - text
  - image
  - audio
  - video

Style:
- Warm
- Calm
- Storybook feeling
- Easy to read

Target:
- Bible stories
- Emotional stories
- Children's stories

System must support:
- Multiple storybooks
- Expandable structure
- AI generation friendly

Design:
- Simple
- Clean
- Mobile friendly
- eBook reading experience
